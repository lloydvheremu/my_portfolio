# OG Image Generation Instructions

The OG image (Open Graph preview image) should be **1200x630px** and will be shown when your portfolio is shared on social media.

## Option 1: Use the HTML template (Recommended)

1. Open `assets/og-image.html` in your browser
2. The page displays a 1200x630px preview
3. Take a screenshot:
   - **Chrome/Edge**: Right-click on the `.og-image` div → "Capture node screenshot"
   - **Firefox**: Use screenshot tool (Ctrl+Shift+S), select the og-image area
   - **Any browser**: Use browser extensions or OS screenshot tools

4. Save as `assets/og-image.png`

## Option 2: Use a design tool

Create in Figma, Canva, or Photoshop with these specs:

- **Dimensions**: 1200x630px
- **Background**: #0a0a0a (near black)
- **Border**: 3px solid #00ff41 with inner glow
- **Main text**: "Lloyd Vheremu"
  - Font: JetBrains Mono Bold
  - Size: 80px
  - Color: #00ff41 (matrix green)
  - Effect: Green glow/shadow
  - Optional: Glitch effect
- **Subtitle**: "Software Developer"
  - Font: Fira Code
  - Size: 36px
  - Color: #e0e0e0
- **Tagline**: "Python · JavaScript · Django · Next.js · Odoo"
  - Font: Fira Code
  - Size: 24px
  - Color: #666666
- **Background effect**: Subtle matrix rain or binary code at 10% opacity
- **Overlay**: Scanline texture for retro terminal feel

## Option 3: Online tool

Use [Canva's OG Image templates](https://www.canva.com/create/og-images/) and customize with your brand colors and text.

## After generation

1. Save the final image as `assets/og-image.png`
2. Verify it's exactly 1200x630px
3. Keep file size under 300KB for fast loading
4. Test how it looks when shared on social media
